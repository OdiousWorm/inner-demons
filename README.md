# inner-demons
Welcome to the README for the Sekiro: Inner Demons gameplay overhaul mod.

It should go without saying, but to play this mod, please download the file-injection tool "Mod Engine" by Katalash. Download can be found here -> https://www.nexusmods.com/sekiro/mods/6?tab=files

Simply copy and paste the included "mod" folder into your Sekiro directory.
